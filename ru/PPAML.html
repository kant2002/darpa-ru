Probabilistic Programming for Advancing Machine Learning (PPAML)

Машинное обучение - способность компьютеров понимать данные, управлять результатами, и выводить понимание из неопределенной информации - это сила за многими современными революциями в вычислениях. Фильтры для email спама, персональные ассистены мобильных телефонов и само-управляемые транспортные средства все основаны на достижениях науки в машинном обучении. К сожалению, несмотря на то что спрос на данные возможности увеличиваются, каждое новое приложение требует Геркулесовых усилий. Команды тяжело находимых экспертов дожлны создавать дорогостоящие, специализированные инструменты которые обычно болезненно медленные и могут вести себя непредсказуемо против больших, сложных наборов данных.

Программа Вероятностного Программирования для Продвижения Машинного Обучения (Probabilistic Programming for Advancing Machine Learning (PPAML) нацелена на решение данных трудностей. Вероятностное программирование это новая парадигма программирования для управления неопределенной информацией. Используя вероятностные языки программирования, PPAML ищет способы значительно увеличить количество людей которые могут успешно строить приложения машинного обучения и сделать экспертов машинного обучения намного более эффективными. Более того, программа ищет способы создать более экономичные, надежные и мощные приложения которым нужно меньше данных для создания более точных результатов - функции недотижимые с текущими технологиями.
