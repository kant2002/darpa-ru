Integrated Cyber Analysis System (ICAS)

Системы разработанные в Интегрированной Системе Кибер Анализа (Integrated Cyber Analysis System (ICAS) сделают судебную информацию о серверах и сети доступной для производственной ИТ безопасности. ICAS автоматически интегрирует структурированные и неструктурированные источники сетевых данных в федерированную базу данных чтобы предоставить возможность рассуждать о киберугрозах, и предоставлять киберзащитникам текущую картину их ИТ окружения, для уменьшения времени требуемого для обнаружения целевых атак. В сердце этой возможности является онтология кибербезопасности которая позволяет автоматическую федерацию нескольких гетерогенных источников данных путем предоставляения единнобразных семантических отношений для защиты ИТ сети.
