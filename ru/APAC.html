Automated Program Analysis for Cybersecurity (APAC)

Программа Automated Program Analysis for Cybersecurity (APAC) нацелена на решение задачи своевременной и надежной проверки безопасности мобильных приложений, сначала определяя свойства безопасности, которые должны быть измерены и затем разрабатывая автоматизированные инструменты для выполнения измерений. APAC сильно отличается от поля официального анализа программ (теория доказывания, логики и машиностроения), чтобы сохранить вредоносный код из рынка приложений DoD Android. APAC стремится применить недавние исследования в этой области в попытке масштабировать возможности DoD анализа программ до уровня, никогда не достигнутого с автоматизированным решением.
